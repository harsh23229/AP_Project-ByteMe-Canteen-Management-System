# AP_Project-ByteMe-Canteen-Management-System

A Java-based canteen management system developed using Swing that simulates a complete food ordering workflow with separate interfaces for customers and administrators. The project follows Object-Oriented Design principles and includes features such as order management, VIP customer handling, persistent file storage, and JUnit-based testing.

Features
Customer and Admin interfaces
Java Swing GUI for menu and order management
VIP and regular customer ordering
Order tracking and history
Persistent file storage using Java File I/O
JUnit test cases for core functionalities
Modular object-oriented architecture using inheritance, interfaces, and encapsulation
Tech Stack
Java
Java Swing (AWT/Swing)
JUnit
File I/O
Collections Framework
